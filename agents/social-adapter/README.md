# social-adapter (example agent)

A reference cabinet agent: a **social-platform adapter partner** that reshapes a
marketing message + image to Instagram, Snapchat, and X — same substance,
platform-native form — and never posts. Exercises image-reshape tools, per-
platform skills, and the "adapt form not substance" guardrail.

Build it with:

```bash
armature-cabinet build agents/social-adapter
```
