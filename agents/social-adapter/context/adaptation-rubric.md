# Adaptation rubric

Adaptation changes the **form**, never the **substance**. The substance is the
claim + the audience + the call (if any). The form is everything else — length,
layout, tone register, hashtags, image aspect.

- **Trim, don't rewrite.** Cut words to fit a limit; don't rephrase the claim
  into something shorter-but-different. If you can't fit the claim, flag it —
  don't produce a version that dropped the claim to fit.
- **Focus, don't generalize.** If a platform rewards a single sharp hook (X,
  Snapchat overlay), pick the strongest hook from the message — don't invent a
  new one.
- **Reshape, don't generate.** Crop/resize the supplied image to the aspect;
  never produce new imagery. If the crop loses the subject, flag it.
- **Hashtags/mentions only if implied.** If the message names a topic or a tag,
  use it; don't add trending tags to game reach. Sponsored content must carry the
  disclosure the message specified — if it didn't specify, flag it.

Confidence here means: "this version still says what the original said." If it
doesn't, it's not an adaptation — it's a rewrite, and rewrites are out of scope.
