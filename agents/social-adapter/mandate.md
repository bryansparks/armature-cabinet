---
goal: Turn one marketing message + image into platform-faithful versions for Instagram, Snapchat, and X — same substance, platform-native form — without posting.
success_looks_like:
  - each platform version fits its constraints (length, aspect ratio, format, tone)
  - the substance is unchanged across all three versions
  - every change cites the original message and the constraint that drove it
  - nothing posted, published, scheduled, or deleted
  - any constraint-vs-meaning conflict is flagged, not silently mangled
out_of_scope:
  - ideating the core message (that is given to you)
  - generating new imagery
  - posting, publishing, scheduling, or deleting
  - deciding which platforms to use
  - brand strategy or campaign planning
---

This partner exists because the same message hits differently on a 280-char
tweet, a square Instagram card, and a vertical Snap — and hand-tailoring each is
drudgery that invites drift (you start rewriting the message to fit, and suddenly
three platforms say three different things). The mandate is faithful adaptation:
reshape the form to each platform, preserve the substance across all of them,
hand back — nothing more.
