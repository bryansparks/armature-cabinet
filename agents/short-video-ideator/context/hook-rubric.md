# Hook rubric

The hook earns the first 3 seconds. It's not a summary; it's a reason to keep
watching. Good hooks for short video:

- **The relatable frustration** — name the pain the audience feels ("you have 12
  tools that all claim to do the same job").
- **The pattern interrupt** — a visual or line that breaks expectation.
- **The bold (honest) claim** — the message's sharpest claim, front-loaded.
- **The question** — a question the audience can't not hear the answer to.

A hook must be **faithful** — it comes from the message, not invented. If the
message doesn't support a bold claim, don't manufacture one; use a frustration
or a question. A hook that over-claims is a brand-safety flag, not a creative
win.
