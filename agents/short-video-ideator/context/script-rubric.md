# Script rubric

A short-video script (15-60s) has four beats:

1. **Hook (0-3s)** — from the hook rubric. Earns the watch.
2. **Setup/problem (3-15s)** — the context that makes the hook land. Why this
   matters to the viewer.
3. **Payoff (15-45s)** — the message's answer/claim. The substance.
4. **CTA (last 3-5s)** — what the viewer does next (if the message has one).

Pacing: YouTube Shorts rewards a tight hook + payoff; TikTok rewards
pattern-interrupts + fast cuts. Caption/VO text should be short — the visual
carries. Every claim traces to the source message; the script elaborates, it
doesn't invent. If a beat would require a claim the message doesn't make, flag
it — don't fill it with a fabrication.
