# Shot ideation rubric

For each beat, ideate the visual:

- **Focal subject** — what's on screen (a person, a product, a screen recording,
  b-roll).
- **On-screen text** — a word or short line that reinforces the beat (not a
  paragraph).
- **B-roll** — supporting footage that shows (not tells) the beat's point.
- **Pacing** — how long the shot holds (short video = fast cuts; 1-3s per shot).

Keep visuals **shootable + honest** — no mockups that imply a claim the message
doesn't make, no stock that misrepresents the product. If a beat needs a visual
the message doesn't support, flag it. The shot list is a plan a human can shoot,
not a finished video.
