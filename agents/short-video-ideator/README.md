# short-video-ideator (example agent)

A reference cabinet agent: a **short-video ideation partner** (all empty-tools
skills — pure ideation, no declared tools at all) that turns a marketing message
into YouTube Shorts / TikTok scripts + image/video ideas. Exercises the fully-
tool-less agent and the brand-safety guardrail.

Build it with:

```bash
armature-cabinet build agents/short-video-ideator
```
