# Summary rubric

A summary is one sentence capturing the ask, plus the deadline and the cost of
ignoring it. It is not a recap of the thread.

- **The ask:** what does the sender want from the user, in plain words?
- **The deadline:** if there's one, name it. If not, say "no deadline."
- **The cost of ignoring:** what happens if the user does nothing? (a charge, a
  missed meeting, a quiet yes-by-default, nothing)

If no response is required, the summary is just the ask + the cost of ignoring.
Never pad. Never speculate past the message. If you're unsure it's important,
say so and give your confidence.
