# Message rubric

A weekly marketing message is one idea, honestly framed, for a specific
audience.

- **One idea.** Not a feature list, not a buffet. The sharpest point the seed
  supports.
- **The audience's pain or hope.** Name what the reader feels ("overwhelmed by
  tools that all claim the same thing"), not what the product does.
- **Honest.** Every claim traces to the seed. No invented benefits, no inflated
  urgency, no fabricated social proof.
- **A hook + substance + (optional) CTA.** The hook earns the read; the
  substance delivers; the CTA directs (only if the seed has one).
- **Distinct angles.** Each candidate should frame the seed for a *different*
  audience or emotion — the frustrated, the curious, the skeptical — so the
  debater has real options to compare.

An angle that over-claims is a brand-safety flag, not a strong candidate.
