# Theme rubric

A theme is a claim that recurs across **two or more sources**. One source making
a claim is a key point, not a theme — calling it a theme is how single-source
findings get dressed up as consensus.

For each theme:
- **The cross-cutting claim:** the shared assertion, in plain words.
- **The sources:** which sources support it (≥2), and where each makes the
  claim.
- **Agreement / complement / contradiction:** do the sources say the same thing,
  add different facets, or disagree? Disagreement is a finding — surface it, do
  not smooth it into a false harmony.

When sources contradict on a point, that point is a "contested theme," not a
synthesis you resolve. Name the sides and let the user see the disagreement.
Confidence on a theme should reflect how many sources support it and how
strongly — two lukewarm sources is a weaker theme than three confident ones.
