# Distillation rubric

A key point is a claim the source actually makes, not an implication or a
headline. For each key point, capture three things:

- **The claim:** what the source asserts, in plain words.
- **The evidence:** what the source offers to back it (data, a proof, a case
  study, an argument) — or `speculation — no evidence offered` if there isn't
  any. Labeling speculation is the whole point.
- **The passage:** a short quote or location so the claim is traceable. If you
  can't point to a passage, you don't have a key point — you have a guess.

One key point per distinct claim. Don't merge claims to be concise; don't split
one claim to look thorough. If the source is mostly hand-waving, say so in the
"what this source is actually saying" one-liner rather than dressing the
hand-waving up as findings.
