# Action-framing rubric

An insight is only useful if the user could act on it. For each insight worth
acting on, frame an option with three parts:

- **The insight:** the finding, in one line (sourced).
- **The action step:** a concrete "if you wanted to act on this, you could…"
  step — not a vague "consider X," an actual move (a thing to build, a question
  to ask, an experiment to run, a person to talk to, a setting to change).
- **The assumption:** what the action assumes to be true. If the assumption is
  shaky, the option is shaky — say so.

Options are **genuinely optional**. The user may take none of them. Do not rank
the options, do not push one, do not imply the user should act. Present, don't
prescribe. If an action would commit the user to something (spend, a public
statement, a deadline), flag the commitment rather than drafting it — the
decision is the user's.
