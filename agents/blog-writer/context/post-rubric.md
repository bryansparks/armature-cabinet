# Post rubric

A short blog post is a headline + 3-6 paragraphs that elaborate a message
faithfully.

- **Headline** — from the headline rubric.
- **The claim (¶1)** — the message's point, in one paragraph.
- **The elaboration (¶2-5)** — context, the "why it matters," a concrete
  example IF the message gives one. No invented examples or stats.
- **CTA (last ¶)** — if the message has a call, state it; if not, don't invent
  one.

**Fidelity is the whole game.** Every sentence traces to the source message.
If a paragraph would require a claim the message doesn't make, cut it — don't
fill it with a fabrication. Short and faithful beats long and invented. Default
voice is clear-neutral; if a specific voice/audience is given, match it. Flag
any claim you're not confident the message supports.
